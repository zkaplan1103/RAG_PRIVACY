# UI-engineer memory

## 2026-06-01 | orchestrator (Phase 0 handoff)

- Entry point: `app.py` at project root. Run with `uv run streamlit run app.py`.
- Import Answer from `src.policylens.generate`. Use `canned_answer()` as the stub.
- In Phase 2 integration the stub call is replaced with real `answer()`.
- Answer schema (docs/CONTRACTS.md §3): `answerable` bool, `text` str, `citations` list[Citation].
  - If `answerable=False`: show a warning banner with the abstention text.
  - If `answerable=True`: show the answer text, then a "Citations" section with expandable cards per citation.
    Each card shows `chunk_id`, `section`, and `quote`.
- Sidebar: policy_id selector (text input or dropdown once real policies are indexed).
- No auth, no multi-user, no fancy styling needed — this is a demo.
